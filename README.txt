-----------------Параметры запука-----------------
Перед началом работы необходимо sv проект или файл вставить в директорию с кодом
Выбор скрипта настраивается в base.json в списке "tasks" (можно поставить true для нескольких одновременно - выполнятся будут последовательно)


Настройка отдельных скриптов согласно заданию выполняется в соответствующих json файлах:

1) Если в json файле в "conf"["allfiles"] стоит true - соответствующий скрипт будет выполнятся для всех sv файлов,
 иначе только для файла (пути к файлу) указанного в "conf"["filename"]

2) В ifdefprocessing.json "includes" - пути к директориям где хранятся включаемые файлы проекта,
                          "defines" - define макросы указанные явно

3) В obfuscator.json "literalclass" - класс обрабатываемых идентификаторов для 2 задания обфускации
                     "module" - название модуля, обрабатываемого для 3 задания обфускации

4) В deobfuscator.json "literalclass" - класс обрабатываемых идентификаторов для 2 задания обфускации
                     "module" - название модуля, обрабатываемого для 3 задания обфускации

5) В read_hierarchy.json "report_filename" - имя создаваемого файла отчета

Обфускация может выполнятся вплоть до нескольких минут в зависимости от обьема проекта
